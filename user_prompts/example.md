# Example Research Task: Renewable Energy in Software Development

## Research Objectives

Investigate the environmental impact and carbon footprint of software development practices, focusing on renewable energy adoption.

## Task Steps

1. Search for academic papers on software sustainability and green computing
2. Find GitHub repositories related to energy-efficient software development
3. Collect data on carbon intensity for major cloud computing regions
4. Map findings to relevant SDG goals (especially SDG 7, 9, and 13)
5. Synthesize the collected information into a brief report

## Expected Outputs

- Literature review summary (3-5 key papers)
- List of relevant open-source projects
- Carbon intensity data table
- SDG alignment analysis
- Executive summary (1-2 pages)

## Keywords

renewable energy, software sustainability, carbon footprint, green computing, energy efficiency
